# Filestream/FileTable with TDE Demo
 Demonstration creating FILESTREAM and FILETABLE with Transparent Data Encryption (TDE).
 
## WARNING: 
EDUCATIONAL USE ONLY. DO NOT USE THIS ON A PRODCUTION SYSTEM. ALWAYS TEST.
